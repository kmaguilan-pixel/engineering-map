About This Project

This is a project initiative by the University of the Philippines - Department of Geodetic Engineering (UP DGE) to create an interactive web mapping application showcasing geospatial data related to UP Diliman. The project aims to provide a user-friendly platform for visualizing and exploring various geospatial datasets.
